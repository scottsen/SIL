## Contributing to Meta Documentation

**These documents should:**
- Be honest and transparent (glass-box principle)
- Focus on decision-making (not just description)
- Include testable hypotheses (empirical grounding)
- Acknowledge uncertainty (no false confidence)
- Update based on new evidence (living documents)

**Writing style:**
- Direct and clear (no academic jargon)
- Bulleted and scannable (busy founders)
- Action-oriented (what to do, not just what to think)
- Evidence-based (cite sources, measurements, examples)
